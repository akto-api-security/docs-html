Burp suite

You can now import API traffic using Akto's burp suite plug-in.
Follow these steps:
  1. 1.
    Go to Quickstart > connect traffic data
  2. 2.
    Navigate to burpsuite.
  3. 3.
    Follow the steps below:
  1. 4.
    You will see all your APIs in the burp collection in API inventory. Whenever you browse your application, all the API calls will populate in the Alkto dashboard> API inventory>burp collection.
